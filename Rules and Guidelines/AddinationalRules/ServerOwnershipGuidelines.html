Rift Labs Studios - Server Ownership Guidelines

Server Types

- Personal Servers: Any non verified server only accesible by invite or direct connect.
- Verified Servers: Dedicated Servers publicly listed at the official Breach Protocol Server Browser.
- Official Servers: Dedicated Servers that are directly managed and operated by Breach Protocol or its authorized representatives.

Basic information about Server Ownership

- Server Ownership is an automatically granted role to anyone who creates a Dedicated Server , Hosts a Server, or is the owner role on a Server for Breach Protocol.
- Ones who own a server have the full responsibility of the server and its members. Including but not limited to rule enforcement, member management, and overall server maintenance. [ Members are still held accountable for breaking the main rules of Breach Protocol even if the server owner fails to enforce them. ]
- Server Owners are expected to maintain a safe and respectful environment for all members, adhering to the core values and guidelines set forth by Breach Protocol.
- Failure to comply with these guidelines may result in consequences, including but not limited to warnings, temporary suspensions, or permanent bans from Breach Protocol. - Players reserve the right to report any server owner or server that violates Breach Protocol's rules and guidelines to the Breach Protocol Moderation Team for review and potential action.
- Breach Protocol reserves the right to take action against any server owner or server that violates its rules and guidelines, including but not limited to removal of server ownership privileges, suspension or banning of the server, or other actions as deemed appropriate by Breach Protocol Moderation Staff.

Server Creation Guidelines

- Servers must be created only with allowed server creation tools provided by Breach Protocol or authorized third-party services such as:
| - The "Host a Server" tool.
| - The official Steamworks dedicated server tool.
- Servers must not be created using unauthorized third-party software or modifications that violate Breach Protocol's terms of service or end-user license agreement.
- Servers must be configured to comply with Breach Protocol's rules and guidelines, including but not limited to content restrictions, player behavior policies, and moderation standards.
- Servers must not be used for any malicious activities, including but not limited to cheating, hacking, or distributing malware.

Moderation Expectations for Server Owners

- Any content that violates the intellectual property rights of others, including but not limited to copyrighted materials, trademarks, and trade secrets must be regulated and prohibited.
- Adult content, pornography, or any materials that are sexually explicit or obscene must be prohibited.
- Hate speech, harassment, or any content that promotes violence or discrimination against individuals or groups based on race, ethnicity, nationality, religion, gender, sexual orientation, or any other characteristic must be prohibited.
- Malware, viruses, or any other harmful software that could damage or disrupt the Breach Protocol infrastructure or its users must be prohibited.
- Any content that is illegal or promotes illegal activities must be prohibited.

Privacy and Security Guidelines

- Server Owners must implement appropriate security measures to protect their servers and user data from unauthorized access, breaches, or other security threats.
- Server Owners must comply with all applicable data protection laws and regulations, including but not limited to the General Data Protection Regulation (GDPR) and the California Consumer Privacy Act (CCPA).
- Server Owners must handle all user data responsibly and as a restricted resource and are prohibited from using it for any unauthorized purposes ( Sharing user data without consent is strictly prohibited ).
- Server Owners must obtain explicit consent from users before collecting or processing any personal data.
- Server Owners must promptly notify game staff and affected users in the event of a data breach or security incident.

Verified Server Guidelines

Verified Servers Basic information

- Verified Servers are Dedicated Servers publicly listed at the official Breach Protocol Server Browser.
- Verified Servers must adhere to additional guidelines and standards set forth by Breach Protocol to ensure a high-quality experience for all players. Listed in this document ( Or any document referenced to ).
- Verified Servers are subject to regular reviews and audits by Breach Protocol to ensure compliance with these guidelines. Failure to comply may result in removal from the Verified Server list or other consequences as deemed appropriate by Breach Protocol.

How to get your server Verified

- To have your server verified and listed on the official Breach Protocol Server Browser, you must meet the following criteria:
| - The server owner must submit a verification request through the official Breach Protocol discord server ( In the intended channel ).
| - The server owner must provide accurate and complete information about the server, including its name, description, rules, and contact information.
| - The server must comply with all Breach Protocol rules and guidelines, including those outlined in this document.
| - The server owner must be over the age of 14.
- Once the verification request is submitted, Breach Protocol will review the server and its information. If the server meets all criteria, it will be added to the Verified Server list and publicly listed on the official Breach Protocol Server Browser.

Naming and Representation Policies

- Verified Servers must have a unique and appropriate name that accurately reflects the server's content and purpose. The name must not contain any offensive, discriminatory, or misleading language.
- Verified Servers must provide a clear and accurate description of the server's content, rules, and guidelines. The description must not contain any false or misleading information.

Responsibility of Verified Server Owners

- Verified Server Owners are responsible for maintaining the quality and integrity of their servers. This includes:
| - Enforcing server rules and guidelines consistently and fairly.
| - Responding promptly to player reports and concerns.
| - Keeping server information up-to-date and accurate.
| - Ensuring that the server remains compliant with all Breach Protocol rules and guidelines.
| - Cooperating with Breach Protocol during reviews and audits of the server.
| - Taking appropriate action in response to any violations of Breach Protocol rules or guidelines by players on their server.
- Failure to uphold these responsibilities may result in removal from the Verified Server list or other consequences as deemed appropriate by Breach Protocol Moderation Staff.

Server Maintenance Requirements

- Server Owners must ensure that their Verified Servers are regularly maintained and updated to provide a stable and enjoyable experience for players.
- We reserve the right to terminate any Inactive , Abandoned, or Neglected Verified Servers from the Verified Server list after a period of 30 days of inactivity.

Community Feedback

- Players are encouraged to provide feedback on Verified Servers to help maintain quality and address any issues that may arise.
- It is prohibited for Verified Server Owners to retaliate against players who provide feedback or report issues related to their servers.

Official Servers

Official Servers Basic information

- Official Servers are Dedicated Servers that are directly managed and operated by Breach Protocol or its authorized representatives.
- Official Servers are intended to provide a high-quality and consistent experience for all players, adhering to the core values and guidelines set forth by Breach Protocol.
- Official Servers are subject to strict rules and guidelines, including those outlined in this document ( Or any document referenced to ).
- Official Servers are regularly monitored and maintained by Breach Protocol to ensure compliance with these guidelines and to address any issues that may arise.
- Notice: Players found exploiting or breaking official game rules while on ( Including but not limited to ) Official Servers are a subject to Global Moderation Actions as they are directly managed by Breach Protocol.
